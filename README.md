# Un Contatore

Da oggi per contare qualsiasi cosa non sarà più necessario utilizzare le dita della mano.


## Description

Un semplice contatore in grado di incrementare tramite il tasto + o decrementare tramite il tasto - un valore di partenza 0.


## Getting Started


### Dependencies

* Windows 10, MacOS


### Installing

* No installation required


### Executing program

* Click on Index.html file
